# Página Responsiva - Grid & Flexbox

Projeto para praticar **CSS Grid**, **Flexbox** e **Media Queries**. Estrutura: header (logo amarela sem bordas arredondadas + menu horizontal), main com cards em grid, footer.

## Tecnologias
HTML5 • CSS3 (Grid, Flexbox, Media Queries)

## Breakpoints (max-width)
- **≤768px (tablet)**: cards em 2 colunas.
- **≤480px (mobile)**: menu em coluna, cards empilhados.

**Justificativa**: 768px cobre tablets (iPad retrato); 480px cobre smartphones. Abordagem desktop-first.

## Recursos Visuais
- Logo em caixa amarela (sem border-radius).
- Cards com hover: borda laranja, sombra, fundo claro e leve elevação.
- Menu com hover laranja.
