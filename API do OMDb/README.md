
# Matriz de Ratings de Episódios (OMDb)

Aplicação web simples que consome a [OMDb API](http://www.omdbapi.com/) para exibir os ratings (notas) dos episódios de uma série de TV, organizados em formato de matriz (tabela), onde cada linha é uma temporada e cada coluna um episódio.

## Funcionalidades

- Busca por nome da série
- Exibição do pôster e título
- Carregamento dinâmico dos episódios de todas as temporadas
- Matriz colorida de acordo com a nota IMDB (verde ≥ 8, laranja ≥ 6, vermelho < 6)

## Como executar

1. Obtenha uma chave de API gratuita em: http://www.omdbapi.com/apikey.aspx
2. Abra o arquivo `script.js` e substitua `'SUA_CHAVE_AQUI'` pela sua chave.
3. Abra o arquivo `index.html` em qualquer navegador moderno.
